# Harunzy Tech Hub Landing Page

A modern, responsive landing page and course registration website for Harunzy Tech Hub.  
Built using pure HTML, CSS, and JavaScript with smooth animations and modern UI effects.

---

## 🚀 Project Overview

Harunzy Tech Hub is a tech training center focused on teaching:

- Graphic Design
- Computer Basics
- Web Development

This landing page allows visitors to:

- Learn about the training center
- View course offerings and pricing (in Ghana Cedis)
- Register for courses online
- Contact the organization easily

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Animated gradient hero section
- Smooth scroll navigation
- Scroll-triggered section animations
- Modern hover effects on course cards
- Styled registration form
- Clean and professional UI
- Pricing in Ghana Cedis (GHS)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- Responsive Design Principles
